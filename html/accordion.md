# Accordion

## Simple accordion
```html
<details>
    This is a simple details content
</details>
```

## Accordion with custom summary
```html
<details>
    <summary>View all details</summary>
    This is a details content with custom summary
</details>
```

## Pre-opened accordion
```html
<details open>
    This is a simple details content
</details>
```
