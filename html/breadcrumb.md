# Breadcrumb

## Breadcrumb navigation
```html
<nav>
    <h1>You are here:</h1>
    <ul id="navlist">
        <li><a href="/">Main</a> →</li>
        <li><a href="/products/">Products</a> →</li>
        <li><a href="/products/dishwashers/">Dishwashers</a> →</li>
        <li><a>Second hand</a></li>
    </ul>
</nav>
```
